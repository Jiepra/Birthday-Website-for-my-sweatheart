Taruh foto-foto kenangan Anda dengan Cipaaa di dalam folder ini!
Format gambar yang disarankan: .jpg, .png, atau .webp (usahakan ukurannya di bawah 1MB agar loading website cepat).

Ganti nama file foto menjadi:
- photo-1.jpg
- photo-2.jpg
- photo-3.jpg
- photo-4.jpg
- photo-5.jpg
- photo-6.jpg

Setelah menaruh file di sini, Anda bisa mengedit data fotonya di file:
`src/data/gallery.js`

Contoh penggunaan di `src/data/gallery.js`:
image: "/images/photo-1.jpg"
